Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET / Read None None A JSON object of all the movies
Returns all current users /users GET N/A N/A A JSON object of all the current users
Returns a specific user by name /users/:userName GET :userName N/A A JSON object of all a specific user
Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user /movies/:title GET / Read :title None A JSON object of one movie
Return data about a genre (description) by name/title (e.g., “Computer-animated”) /movies/genres/:genreName GET / Read :genreName None returns all movies with the genre searched
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET / Read :directorName None A JSON object of a especific director
